# Netflix-Dataset-Vsualization-Pandas-Matplotlib
Netflix dataset visualization project
<br>

This project analyzes and visualizes the Netflix dataset using Python. The goal is to explore trends, content distribution, movie durations, ratings, release years, and other insights through data analysis and visualization.

Tools & Technologies Used
>Python
>Pandas
>Matplotlib
>Jupyter Notebook

>Project Objectives
>Clean and preprocess Netflix dataset
>Analyze movies and TV shows
>Visualize content trends using charts and graphs
>Practice data analysis and visualization skills
